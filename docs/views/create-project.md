# 创建Vite2的项目

## npm创建
``` sh
npm init @vitejs/app
```

## yarn创建
``` sh
yarn create @vitejs/app
```

## 创建项目时指定名称和模板
``` sh
npm init @vitejs/app my-vue-app --template vue
```
## 选用选择模板的方式创建项目
- npm init @vitejs/app
- 选择vue
- cd vite-project
- npm install
- npm run dev
